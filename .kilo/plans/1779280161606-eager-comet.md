# Logo Unification Plan

**Date:** 2026-05-20  
**Scope:** Extract a shared `Logo` component to unify the logo icon, text, and styling across all pages

---

## Discrepancies Identified

Four locations display a logo in the left side of the navigation bar, but they use **two different designs**:

| Location | Icon | Text | Has Link? |
|---|---|---|---|
| `Navbar.tsx` (transfer + legal pages) | `<Network>` lucide icon | UPPERCASE `SENDME.ALT` | Yes (`<Link href="/">`) |
| `DesktopLanding.tsx` | Custom "S" letter in rounded box | lowercase `sendme.alt` | No |
| `TabletLanding.tsx` | Custom "S" letter in rounded box | lowercase `sendme.alt` | No |
| `MobileLanding.tsx` | Custom "S" letter in rounded box | lowercase `sendme.alt` | No |

The landing pages (3/4 locations) use a consistent design; the Navbar (1/4) diverges.

**Chosen unified design:** "S" box icon + lowercase `sendme.alt` text (user confirmed).

---

## Implementation Steps

### Step 1: Create shared `Logo` component

**File:** `src/components/Logo.tsx` (new)

- Icon: Styled `<div>` with "S" letter, responsive sizing controlled by a `size` prop
- Text: lowercase `sendme.alt` with tracking-tight
- Props: `size?: 'sm' | 'md' | 'lg'`, `withLink?: boolean`
- When `withLink` is true, wraps in `<Link href="/">` with `hover:opacity-80`
- Sizing table:

| size | Icon container | Icon font |
|---|---|---|
| `'sm'` | `w-7 h-7 rounded-md` | `text-xs` |
| `'md'` | `w-8 h-8 rounded-lg` | `text-sm` |
| `'lg'` | `w-10 h-10 rounded-xl` | `text-lg` |

### Step 2: Replace inline logos (4 files)

| File | Line(s) | Replace with |
|---|---|---|
| `layout/Navbar.tsx` | 8–10 | `<Logo size="sm" withLink />` |
| `DesktopLanding.tsx` | 12–14 | `<Logo size="lg" />` |
| `TabletLanding.tsx` | 12–14 | `<Logo size="md" />` |
| `MobileLanding.tsx` | 12–14 | `<Logo size="sm" />` |

### Step 3: Remove unused import

- `Navbar.tsx`: Remove `import { Network } from 'lucide-react'` (no longer needed)

### Step 4: Verify

- `npx tsc --noEmit` — typecheck
- `npm run build` — production build compiles
